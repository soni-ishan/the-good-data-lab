# The Good Data Lab

**Code. Data. Community.**

A beginner-friendly space for undergrads to explore Data Science & AI. We build projects that do good, share ideas, and learn together.

🌐 **Live Site:** [the-good-data-lab.vercel.app](https://the-good-data-lab.vercel.app)

💬 **Join Us:** [Discord Server](https://discord.gg/w4YuB5QY3u)

## About

The Good Data Lab is a student-run collective tackling real-world challenges through data science and AI. From Kaggle competitions to building agentic AI systems, we bridge the gap between academic theory and real-world impact.

### Featured Projects

- **Course Compass** — A specialized chatbot for university CS students powered by a Neo4j Knowledge Graph
- **CSIRO Image2Biomass Prediction** — ConvNeXt-based Deep Learning pipeline for vegetation biomass estimation
- **Microsoft AI Dev Days Hackathon** — Currently participating!

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Analytics:** [Vercel Analytics](https://vercel.com/analytics) + [Speed Insights](https://vercel.com/docs/speed-insights)
- **Deployment:** [Vercel](https://vercel.com/)

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/soni-ishan/the-good-data-lab.git
cd the-good-data-lab

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
